# SustainScore

SustainScore is a student project for the **Principles of Software Development** course at the University of Luxembourg.

The goal of the project is to design, build, test, and evaluate a small but complete mobile application while applying software development principles such as requirements engineering, design, implementation, CI/CD, testing, and project documentation.

## Project Overview

SustainScore is a mobile application that helps users become more aware of their daily resource consumption habits. Users will be able to log predefined activities, view a sustainability score, compare their behavior with a recommended profile, and receive improvement suggestions.

## Key Features

- Activity logging for predefined resource-related actions.
- Sustainability score calculation.
- Visual comparison between actual and recommended behavior.
- Daily and weekly progress summaries.
- AI-generated feedback with actionable suggestions.

## Project Phases

The project follows the course delivery phases:

1. Idea and scoping
2. Requirements
3. Cost and complexity analysis
4. Design
5. Implementation
